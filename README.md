### About

I'm an AI-native software engineer. I ship fast, across the full stack, and I contribute upstream to projects that real infrastructure runs on.

I work on distributed systems, AI/ML tooling, and developer infrastructure. I build with TypeScript, Python, Java, Go, and C/C++. I live inside agentic dev tools (Claude Code, Codex, Cursor, Windsurf, Devin, Zed) and compose them into pipelines that ship production software faster than most teams. Some people call it vibe engineering. I call it leverage.

Currently:

- **Google Summer of Code 2026** with the **Apache Software Foundation**, building a portable Kafka Streams runner for Apache Beam.
- **Kubernetes Release Team** as a **Release Signal Shadow** (v1.37 cycle).
- Shipping side projects and writing about LLM tooling and agentic AI on [my blog](https://junaidshaukat.com/blog).
